# scientific-illustration


这种一般叫：

**论文方法结构图 / 模型架构示意图**
英文常见叫法是：

* **model architecture diagram**
* **pipeline diagram**
* **method overview figure**
* **framework illustration**

如果更细一点描述你这张图的视觉风格，可以说是：

**CV/AI 论文里的扁平化信息图风格的 pipeline schematic**

它的典型特征是：

* **扁平化矢量风格**，不是写实插画
* **模块框 + 箭头流向** 的流程表达
* 用**浅蓝、灰蓝、浅黄、浅绿**这类低饱和科研配色
* 有一点 **pseudo-3D / isometric-ish** 的层叠卡片效果
* 强调**结构清晰、信息分区、可发表性**，而不是装饰性

如果你要给设计师或 AI 作图工具下指令，比较准确的说法可以写成：

**“CVPR/ICCV 风格的科研论文模型结构图（pipeline diagram / architecture figure），扁平化矢量信息图风格，带轻微 3D 层叠效果。”**

再细分一点，这张图还带有：

* **academic infographic**
* **technical schematic**
* **neural network architecture visualization**

不是下面这些：

* 不是海报风
* 不是 Nature 那种重质感科普插图
* 不是纯 UML / 工程流程图
* 也不是生物医学那种 cell/organ 渲染图

你要的话，我还能帮你把这种风格整理成一套可复用的提示词模板，专门用于生成“顶会论文风格结构图”。
